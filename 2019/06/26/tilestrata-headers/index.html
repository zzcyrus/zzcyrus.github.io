4. 代码浅析

原理比较简单，在tilestrata的设计中，会使用use去注册各种类型的插件，tilestrata-headers就是reshook类型的插件，在初始化的过程中会被注册

通过这种方式，插件劫持每个 response，给每个 response header 加上我们自定义的内容。
