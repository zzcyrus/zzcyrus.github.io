tilestrata-blend 插件使用及浅析

2019-07-08
  • TileStrata

tilestrata-blend插件可以将多个栅格瓦片图层合并成一个 png 图层，并且支持透明度、混合模式、图层过滤等选项。使用这个插件必须安装依赖node-mapnik。这个插件依然是一个利用 mapnik 强大的功能的Provider类型的插件，甚至可以基于 mapnik 增强它已有的功能。

demo 项目可以参考tilestrata-sample-code

4. 代码浅析

这个插件在背后依赖了tilestrata-dependency，所以配置里面的['world_merc', 'tile.png']实际上是通过执行了.use(dependency('world_merc_png', 'tile.png'))操作来获取数据源：

当请求某个瓦片时候，对每个layers依次执行fetchTile获取对应图片和prepareImage预处理图片

完成每个数据源的单独处理后就是合并操作了，利用 mapnik 的omposite方法来依次合并创建的每个mapnik.Image对象，最后再利用demultiply解构为 buffer 返回给 tilestrata 就好了。
